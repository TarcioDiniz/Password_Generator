# Password_Generator

A small project to train the concepts of HTML, CSS & JAVASCRIPT.
It's very simple, run index.html and enjoy generating completely random passwords.
Choose defaults and generate passwords as you like:

## Features

- CHARACTER SIZE
- CAPITAL LETTER
- LOWER CASE
- NUMBERS
- SYMBOLS
